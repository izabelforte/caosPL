A história por trás de tudo

O Flappy Bird, jogo que inspirou o projeto "Caos no Polo Norte", foi criado pelo desenvolvedor vietnamita Dong Nguyen e lançado em maio de 2013. O jogo tornou-se incrivelmente popular devido à sua simplicidade e dificuldade desafiadora.

O Jogo Caos no Polo Norte foi idealizado pelos formandos Camilla Nunes, Izabel Forte, João Pereira e Lybia Marçal para apresentação como trabalho final da UFCD 10794 Python Avançado, sob instruções do formador Manuel Marinho.
O projeto foi desenvolvido em Python, utilizando a biblioteca Pygame, o framework Django e códigos HTML e CSS.

Pygame é uma biblioteca de jogos em Python que permite a criação de jogos de forma relativamente simples, enquanto Django é um framework web de alto nível que facilita a construção de aplicativos web robustos e escaláveis.
